## âŒ Common Issues & Solutions

### Can't See Notes Tab?
- Refresh the page
- Make sure you're in a job detail page
- Check if you have admin access

### Reminder Not Saving?
- Click "Create Reminder" button
- Check that message is not empty
- Verify time is set

### Task Won't Reassign?
- Select task from dropdown
- Select team member from dropdown
- Click "Assign Task" (not "Change Assignment")

### Equipment Not Appearing?
- Click "Add Equipment" button
- Fill all required fields
- Click "Add Equipment" to save
- Refresh if needed

### Permit Expiry Warning?
- Check "Expiry Date" column
- Set renewal 30 days before
- Update status when renewing
